TROUBLESHOOTING
==============================================================================

ISSUE: "fatal: not a git repository"
SOLUTION: 
  - Ensure you're in correct directory
  - Run: cd c:\Users\Ayaan\Desktop\Project-Zeb\versions_history\v1.1\version_1
  - Check if .git folder exists (dir .git)

ISSUE: "rejected" error when pushing
SOLUTION:
  - Pull latest: git pull --rebase origin versions_history/v1.1
  - Try push again: git push origin versions_history/v1.1

ISSUE: .env file is showing to commit
SOLUTION:
  - .env should already be ignored by .gitignore
  - If already committed: git rm --cached .env
  - Then commit and push

ISSUE: Cannot tag - "tag already exists"
SOLUTION:
  - Use different tag name: v1.1-final
  - Or delete existing: git tag -d v1.1 (then git push origin --delete v1.1)

ISSUE: "fatal: Your current branch does not have any commits yet"
SOLUTION:
  - This is initial commit
  - Just proceed with git add and commit

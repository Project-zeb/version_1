## TROUBLESHOOTING FOR USERS

### "Can't connect to database"
- Check .env has correct DB_HOST, DB_USER, DB_PASSWORD
- Verify database server is running
- Ensure database exists (create with database_setup.sql)

### "Module not found"
- Run: `pip install -r requirements.txt`
- Verify virtual environment is activated
- Check Python version is 3.8+

### ".env not found"
- Create: `cp .env.example .env`
- Edit with real credentials
- Must be in projectz_v.2/ directory

### "Permission denied" on first run
- Ensure app.db location is writable
- Check database user has correct privileges
- Run with appropriate permissions
